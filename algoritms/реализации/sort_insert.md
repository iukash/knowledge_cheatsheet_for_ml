```python
def sort_insert(nums: list) -> list:
    for i in range(1, len(nums)):
        key = nums[i]
        while nums[i-1] > key and i > 0:
            nums[i] = nums[i-1]
            i -= 1
        nums[i] = key
    return nums
```
